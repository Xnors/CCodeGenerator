# CCodeGenerator

## 简介

这是一个类似于 [`LLVM`](https://llvm.org/), 使用 `Rust` 程序接口生成 `C 语言`代码的库

## 项目结构

```
.\
│  .gitignore
│  Cargo.lock
│  Cargo.toml
│  README.md
│
├─.github
│  └─workflows
│          rust-auto-test.yml
│
├─src
│  │  lib.rs
│  │
│  └─core
│          context.rs
│          mod.rs
```

> 注意:
> 包名为 `ccgenor` 而非 `c_code_generator`
